# Catálogo de Produtos

Projeto de um catálogo para cadastro, consulta e alteração de produtos.

## Ferramentas utilizadas:
- PHP 7.0
- Laravel 5.2
- MySQL 5.7.18
- HTML5, CSS3 e jQuery
- Bootstrap
